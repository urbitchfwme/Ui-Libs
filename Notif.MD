# Notfication Library



## Loadstring
```lua
local NotifyLibrary = loadstring(game:HttpGet("https://raw.githubusercontent.com/Kinlei/Dynissimo/main/Scripts/AkaliNotif.lua"))()
local Notify = NotifyLibrary.Notify
```

## Sending Notif
```lua
Notify({
        Title = "Title",
        Description = "Descreption"
        Duration = 3
})
```
